# ToDo App Backend

## Getting Started

1. Prerequisite:

   - MySQL Server
   - Node.js

2. Create DB:

   - Update DB Authentication details in `./config/config.json`

   ```
   Create Database 'todo-app'
   ```

3. Install Dependancies

   ```
   $ npm install
   ```

4. Run Locally

   - By default backend runs at http://localhost:8080

   ```
   $ npm start
   ```

- Default Login Credentials:

  ```
  Admin:
      Email: admin@todo.lk
      Password: abc123
  Guest:
      Email: guest@todo.lk
      Password: abc123
  ```
